# crystal-todo

A simple example of a todo list in Crystal

## Installation
### Prerequisites

Using LevelDB to store list and requires 
https://github.com/crystal-community/leveldb

Debian:

```
sudo apt-get install libleveldb-dev libleveldb1v5 libsnappy1v5
```
